NAT

  • Host port the port in your physical machine ,which you want to open
  • Virtual machine IP address your virtual machine IP address
  • Virtual machine port the port ,which you want to mapping

For example , your Host ip is 192.168.1.104, and your Virtual Machine ip is 10.16.40.100

the port in your VM will be translated to host Specified port

10.16.40.100:3306==>192.168.1.104:45565

In addition to this, you should close your VM firewall.

Add Inbound Rule

Ok, if your phone and pc are in the same network , type the pc ip and port, you can access the port in VM.
