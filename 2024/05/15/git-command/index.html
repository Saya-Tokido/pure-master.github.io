branch

分支介绍

一般的，开发过程中有三种典型分支 main develop feature

  • main 分支是正式上线的分支，开发时不使用
  • develop 功能开发完成后合并到该分支
  • feature 开发时最常用的分支，不同于前两者，该分支在不需要时会被删除，完善后被合并到develop分支
