创建子网

手动创建一个子网，用于各服务之间通过服务名直接通信

  • driver 是指定连接模式，默认桥接
  • subnet指定子网ip范围

准备docker-compose启动配置文件

编写docker-compose.yml

  • version 版本号，高版本dockerc-ompose不指定也可
  • services 下列出有哪些服务，以及容器启动方式
    • mysql 是服务名称，可以自行指定，服务间通过服务名称进行网络通信
      • image指定镜像，如果本地不存在会去镜像仓库拉取
      • environment 环境变量
      • volumes 数据卷挂载形式 <宿主机目录>:<容器内目录>
      • networks 指定容器网络信息
        • appsubnet 容器所处子网名称
          • ipv4_address 指定容器所处子网占用的ipv4地址
      • ports 指定容器的端口映射，将容器内的ip:port映射到宿主机的ip:port用于外部通过宿主机ip和指定端口访问容器
      • command 指定容器启动时的执行命令
  • networks 指定服务网络配置
    • appsubnet 子网名称
      • external 表示使用已经创建的子网，不需要再创建
        • name 外部使用的子网名称
