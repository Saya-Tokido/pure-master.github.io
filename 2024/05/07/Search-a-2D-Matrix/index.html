Description

You are given an m x n integer matrix matrix with the following two properties:

  • Each row is sorted in non-decreasing order.
  • The first integer of each row is greater than the last integer of the previous row.

Given an integer target, return true if target is in matrix or false otherwise.

You must write a solution in O(log(m * n)) time complexity.

Constraints:

  • m == matrix.length

  • -104 <= matrix[i][j], target <= 104

代码讲解

upper_bound是cpp STL中的函数，用于在已排序的区间中查找第一个大于某个值的元素的位置

lambda 表达式 [](const int b, const vector<int> &a) { return b < a[0]; } 是用作比较函数。这个 lambda 表达式接收两个参数：b 是我们要查找的目标值 target，a 是二维矩阵的每一行。lambda 表达式返回 true 表示第一个参数小于当前行的第一个元素，否则返回 false。这样做的目的是让 std::upper_bound 函数根据每行的第一个元素进行比较。

如果找出来第一行元素大于target(即row==matrix.begin())，说明target比矩阵中所有元素都小，返回false

如果row==matrix.end()说明两种可能

  1. target在最后一行元素的范围中
  2. target比所有元素都要大

二分查找可以同时解决两种可能
