usage

  1. 使用 l<r

    • 当你确定搜索范围时，右边界 r 的值通常是数组长度或列表长度减去 1，而左边界 l 的值通常是 0。

    • 在每次迭代时，你都会将搜索范围缩小为 [l, mid) 或 [mid+1, r)，因此当 l 和 r 相等时，搜索范围为空，此时应该停止搜索。

    • 由于每次迭代都排除了 mid，因此 l 最终会超过 r，这时搜索结束。

  2. 使用 l<=r

    • 当你确定搜索范围时，左边界 l 和右边界 r 都是有效的索引。

    • 在每次迭代时，你会将搜索范围缩小为 [l, mid-1] 或 [mid+1, r]，此时包含了 mid。

    • 当 l 和 r 相等时，仍然需要继续搜索，因为此时可能 l 或 r 就是要找的值。
