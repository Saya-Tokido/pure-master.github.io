introduction

if you have a class Bird

Some class, such as Woodpecker,Hummingbird may extends this class

However,there are some class ,like Ostrich DodoBird ,which can’t fly.

So we should override the fly method.

It’s a solution, but you need to override the fly method many times,just to do one thing: not fly,if there are many birds can’t fly.

How to simplify?

Solution

You can define a interface FlyBehavior

Then implements FlyBehavior interface

Then class Bird can make it its own property.

Now, just pass the Flyable or FlynoWay,you can complete all the class extends Bird .

I think this is a solution to Java without multiple inheritance.

strategy pattern (also known as the policy pattern) is a behavioral software design pattern that enables selecting an algorithm at runtime. Instead of implementing a single algorithm directly, code receives runtime instructions as to which in a family of algorithms to use.
